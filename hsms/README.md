# HSMS Monorepo

## Structure

- `backend/`: Node.js + Express + JSON database
- `frontend/`: React + Router + Axios + Bootstrap

## Run (development)

```bash
npm install
npm run dev
```

- Backend: `http://localhost:4000`
- Frontend: `http://localhost:3000`
